## 2023-Present Frontend Chapter Lead · 2019-2023 Frontend Developer

# DFDS A/S

At DFDS I run the Frontend Chapter for the Ferry tribe, setting frontend direction and standards across teams while staying hands-on. Across my time here I led the rewrite of the Freight B2B platform onto a microservice backend: a Next.js multi-zone monorepo (Control Tower) on the frontend and a BFF (Backend for Frontend) in Nest.js, alongside on-and-off maintenance of the legacy .NET MVC solution. More recently I built AI document ingestion in MyFreight, turning unstructured shipping documents into structured data.

We run a Serverless setup with a DevOps flow in GitHub Actions, with automated tests in Jest and Playwright. As Chapter Lead I also set frontend direction and standards across teams, and increasingly build with an AI-assisted, agentic workflow.
